Les variables globals es poden modificar

A banda de consultar el seu valor, una variable global també pot ser modificada des dels blocs pels que és global.

Per exemple, recordem la següent versió del programa ComptaAiEs:

 1public class ComptaAiEs {
 2    public static void main(String[] args) {
 3        System.out.println("Introdueix un text");
 4        String entrada = Entrada.readLine();
 5        int numAs = quantesOcurrencies(entrada, 'a');
 6        int numEs = quantesOcurrencies(entrada, 'e');
 7        mostraOcurrencies('a', numAs);
 8        mostraOcurrencies('e', numEs);
 9        mostraComparacio('a', numAs, 'e', numEs);
10    }
11    public static void mostraOcurrencies(char lletra, int quantes) {
12        System.out.println("Nombre de '" + lletra + "'s: " + quantes);
13    }
14    public static void mostraComparacio(char lletra1, int quantes1, char lletra2, int quantes2) {
15        if (quantes1 > quantes2) {
16            System.out.println("Hi ha més '" + lletra1 +"'s que '"+lletra2+"'s");
17        } else if (quantes1 < quantes2 ) {
18            System.out.println("Hi ha menys '" + lletra1 +"'s que '"+lletra2+"'s");
19        } else {
20            System.out.println("Hi ha tantes '" + lletra1 +"'s com '"+lletra2+"'s");
21        }
22    }
23    public static int quantesOcurrencies(String text, char lletra) {
24        int comptador = 0;
25        for (int i=0; i < text.length(); i++) {
26            if (text.charAt(i) == lletra) {
27                comptador += 1;
28            }
29        }
30        return comptador;
31    }
32}

Considera ara la següent versió del mateix programa:

 1public class ComptaAiEs {
 2    private static int quantesAs = 0;
 3    private static int quantesEs = 0;
 4
 5    public static void main(String[] args) {
 6        System.out.println("Introdueix un text");
 7        String entrada = Entrada.readLine();
 8        comptaAiEs(entrada);
 9        mostraOcurrencies('a', quantesAs);
10        mostraOcurrencies('e', quantesEs);
11        comparaOcurrenciesAiEs();
12    }
13    public static void mostraOcurrencies(char lletra, int quantes) {
14        System.out.println("Nombre de '" + lletra + "'s: " + quantes);
15    }
16    public static void comparaOcurrenciesAiEs() {
17        if (quantesAs > quantesEs) {
18            System.out.println("Hi ha més 'a's que 'e's");
19        } else if (quantesAs < quantesEs ) {
20            System.out.println("Hi ha menys 'a's que 'e's");
21        } else {
22            System.out.println("Hi ha tantes 'a's com 'e's");
23        }
24    }
25    public static void comptaAiEs(String text) {
26        for (int i=0; i < text.length(); i++) {
27            if (text.charAt(i) == 'a') {
28                quantesAs += 1;
29            } else if (text.charAt(i) == 'e') {
30                quantesEs += 1;
31            }
32        }
33    }
34}

Aquesta nova versió ha reemplaçat la funció quantesOcurrencies() per un procediment que compta a l'hora les ocurrències de 'a' i de 'e'. Com que una funció no pot retornar més d'un valor, per poder obtenir els dos valors a l'hora, una manera és anar dipositant els càlculs a una variable global.

Atenció: la modificació de variables globals sol ser considerada una pràctica pobre de programació que, sovint, porta problemes. De moment se'ns perdona fer-ho servir fins que no descobrim alternatives.
