Exercici 34_25. La classe Hora

Context

  • Carpeta de lliurament: 34_25_hora/

  • Continguts relacionats: Constructors

  • Com lliurar-lo: instruccions

  • [✓] Exercici amb autoavaluació

Enunciat

Implementa la classe Hora amb les propietats (privades) hora, minut i segon de tipus enter, i els accessors (públics) que assegurin que l'hora és sempre vàlida.

Per defecte, l'hora correspondrà a 0:00:00, però la classe oferirà els mecanismes necessaris per a inicialitzar una hora amb algun altre valor vàlid. És a dir, oferirà un constructor específic amb la signatura Hora(int hores, int minuts, int segons) que, en cas de que els valors rebuts no corresponguin amb una hora vàlida, inicialitzarà l'hora amb el valor per defecte.

Afegeix els següents mètodes:

  • void incrementa(): fa que l'hora tingui un segon més

  • void incrementa(int): fa que l'hora s'incrementi en el nombre de segons indicats

  • void decrementa(): fa que l'hora tingui un segon menys

  • void decrementa(int): fa que l'hora es decrementi en el nombre de segons indicats

  • int compareTo(Hora): compara amb l'hora indicada i retorna <0 si és menor que la indicada, 0 si són iguals i >0 si és més gran que la indicada.

  • String toString(): retorna un String amb l'hora en format h:mm:ss.

Inclou a Hora.java el següent main()` per fer una prova senzilla:

 1/**
 2 * Compara dues hores i retorna l'operador corresponent
 3 * Per exemple, si hora1 és menor que hora2, l'operador serà "<". Els
 4 * altres dos valors possibles són ">" i "=="
 5 * @param hora1: primera hora a comparar
 6 * @param hora2: segona hora a comparar
 7 * @return operador resultant
 8 */
 9private static String composaOperadorComparacio(Hora hora1, Hora hora2) {
10    int comparacio = hora1.compareTo(hora2);
11    if (comparacio < 0) {
12        return "<";
13    } else if (comparacio > 0) {
14        return ">";
15    } else {
16        return "==";
17    }
18}
19
20public static void main(String[] args) {
21    Hora hora1 = new Hora();
22    Hora hora2 = new Hora(0, 0, 2);
23    System.out.printf("Inicialment hora1: %s %s hora2: %s%n",
24            hora1,
25            composaOperadorComparacio(hora1, hora2),
26            hora2);
27    System.out.println("Incrementem 1 segon a la primera i decrementem 1 segon a la segona");
28    hora1.incrementa();
29    hora2.decrementa();
30    System.out.printf("Finalment hora1: %s %s hora2: %s%n",
31            hora1,
32            composaOperadorComparacio(hora1, hora2),
33            hora2);
34
35}

Amb aquest main() l'execució de Hora generarà:

$ javac Hora.java && java Hora
Inicialment hora1: 0:00:00 < hora2: 0:00:02
Incrementem 1 segon a la primera i decrementem 1 segon a la segona
Finalment hora1: 0:00:01 == hora2: 0:00:01

Notes / pistes

  1. En cas que un constructor específic es trobi amb un o més paràmetres no vàlids, inicialitzarà la instància amb els valors per defecte: 0:00:00.

  2. Per escriure el 0 inicial als minuts i segons quan el valor és menor de 10, es pot fer servir el mètode System.out.format()

    Per si vas amb el temps molt just:

    String.format("%d %02d %02d", 4, 4, 4)     // 4:04:04

  3. Quan hi ha tants jocs de prova com en aquest exercici, si no estàs fent servir un IDE és possible que et costi trobar els errors. De vegades pot ser útil redireccionar la sortida de prgtest. Per exemple:

    $ prgtest | less

$ prgtest &> /tmp/resultat ; less /tmp/resultat

    Et permetrà navegar per la sortida de prgtest. Pots sortir prement q.
