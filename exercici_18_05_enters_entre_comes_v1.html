Exercici 18_05. Enters entre comes (v1)

Context

  • Carpeta de lliurament: 18_05_enters_entre_comes_v1/

  • Continguts relacionats: Seqüències en Java

  • Com lliurar-lo: instruccions

  • [✓] Exercici amb autoavaluació

Enunciat

Desenvolupa un programa que mostri tres enters separats per comes tenint en comptes que els tres valors es troben emmagatzemats en un array d'enters.

Et passo una plantilla on hauràs de reemplaçar els valors que apareixen /*…*/ pel contingut corresponent:

 1/*…**/
 2public class EntersEntreComes {
 3    public static void main(String[] args) {
 4        int[] numeros;
 5        numeros = new int[3];
 6        numeros[/*…*/] = 1;
 7        numeros[/*…*/] = 2;
 8        numeros[/*…*/] = 3;
 9        System.out.print(numeros[/*…*/]);
10        for (int i = 1; i < numeros/*…*/; i++) {
11            System.out.print(", " + numeros[/*…*/]);
12        }
13        System.out.println();
14    }
15}

La sortida d'aquest programa hauria de ser la següent:

1, 2, 3
