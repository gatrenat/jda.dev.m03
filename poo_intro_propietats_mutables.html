Propietats mutables

Podem canviar el nombre de vides del gat Renat simplement afegint les següents línies al final del main():

 1public class GatRenat {
 2    int vides = 7;                // vides disponibles del gat Renat
 3    public static void main(String[] args) {
 4        GatRenat renat;           // declarem l'objecte (la referència) al gat
 5        renat = new GatRenat();   // creem la instància del gat Renat.
 6        System.out.println("Abans el gat Renat tenia " + renat.vides + " vides");
 7        renat.vides = renat.vides - 1;  // Renat ha tingut un accident
 8        System.out.println("Ara el gat Renat té " + renat.vides);
 9    }
10}

Modifica la classe GatRenat amb els canvis, executa el codi i contesta les següents preguntes:

  1. Com podem consultar (llegir, obtenir) el valor d'una propietat de la instància?

  2. Com podem canviar (escriure, assignar) el valor d'una propietat de la instància?

  3. És possible indicar que el gat Renat té -12 vides? Com? És quelcom desitjable que pugui passar? En cas que es pugui, com ho podries evitar fent servir els coneixements de programació estructurada i modular que hem treballat fins ara?
