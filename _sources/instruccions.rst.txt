########################
Instruccions seqüencials
########################

Un programa no deixa de ser una seqüència d'instruccions que
l'ordinador va executant una darrera de l'altra.

Podem classificar les instruccions bàsiques en tres categories:
seqüencials, condicionals i repeticions.

.. uml::
    :align: center
    :caption: un primer desglossament de les instruccions

    @startuml

    hide circle
    hide empty members
    skinparam monochrome true

    class Instruccions
    class Seqüencials extends Instruccions
    class Condicionals extends Instruccions
    class Repeticions extends Instruccions
    class Declaracions extends Seqüencials
    class Assignacions extends Seqüencials
    class EntradaSortida extends Seqüencials

    @enduml


En aquesta secció coneixerem les instruccions seqüencials. Les
instruccions seqüencials les classificarem en tres: declaracions,
assignacions i d'entrada/sortida, i només amb aquestes, ja podrem
desenvolupar els nostres primers programes.
