.. rubric:: Enunciat


Desenvolupa una nova versió de l'exercici que :doc:`dibuixava un triangle
de lletres <exercici_15_09_trianglelletres>`. 

La nova versió tindrà la mateixa sortida que l'original però el codi del
programa serà més modular. En concret, el mòdul ``main()`` se
se n'encarregarà d'obtenir les dades d'entrada, i cridarà un
nou mòdul anomenat ``dibuixaTriangle()`` que serà qui realitzi realment
la feina de mostrar el triangle. De fet, aquest es recolzarà en un altre
anomenat ``dibuixaLinia()`` que serà qui realment faci la feina de
dibuixar cada línia del triangle.

La signatura dels nous mòduls serà:

.. code-block:: java

    public static void dibuixaTriangle(String text)
    public static void dibuixaLinia(String text, int linia)
