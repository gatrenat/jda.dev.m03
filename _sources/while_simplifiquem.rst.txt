############
Simplifiquem
############

Quan el problema ens resulta molt complex, una de les estratègies que
podem considerar és començar per un problema més fàcil.

Comencem, doncs, simplificant el problema. Suposem que només hem de llegir un
valor. En aquest cas, el programa podria ser així de fàcil:

.. code-block:: java
    :linenos:

    public class SumaPositius {
        public static void main(String[] args) {

            // creem la variable on guardarem el resultat de sumar
            int suma = 0;   // inicialment no hem sumat res i per tant és 0

            // obtenim el valor
            System.out.println("Introdueix un valor");
            int valor = Integer.parseInt(Entrada.readLine());

            // realitzem les comprovacions
            if (valor >= 0) {
                suma = suma + valor;
            }

            // mostrem el resultat
            System.out.println("La suma és " + suma);
        }
    }

Hi ha maneres més fàcils de fer-ho, ja sé, però aquesta ens permetrà
acostar-nos a la solució definitiva del nostre problema.

De moment fixa't que ``suma`` inicialment val 0 (per la línia 5) i a la
línia 13 afegirem al zero inicial el valor llegit, sempre i quan aquest
sigui positiu, és clar.
