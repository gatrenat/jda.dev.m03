#########################
Llenguatge de programació
#########################

En aquest curs farem servir el llenguatge de programació Java. En concret,
farem servir una implementació anomenada *OpenJDK* en la seva versió 17.

Si tens qualsevol altra implementació o/i versió de Java ja instal·lada, sempre
que la versió sigui >=1.17) no hauries de tenir cap problema per seguir el
curs. En cas que la versió sigui anterior, molt probablement et trobaràs que no
podràs realitzar algunes activitats amb normalitat.

Comprova la teva versió amb les comandes:

.. code-block:: console
    :emphasize-lines: 2, 4-

    $  javac -version
    javac 17.0.4
    $  java -version
    openjdk version "17.0.4" 2022-07-19
    OpenJDK Runtime Environment (build 17.0.4+8-Debian-1deb11u1)
    OpenJDK 64-Bit Server VM (build 17.0.4+8-Debian-1deb11u1, mixed mode, sharing)

El teu resultat no té perquè ser idèntic als meus. Únicament t'has
d'assegurar que ha trobat la comanda i que la versió és 1.17 o més.

Si no tens instaŀlat Java, et generarà un missatge similar a:

.. code-block:: console
    :emphasize-lines: 2

    $ javac -version
    bash: javac: no s'ha trobat l'ordre

En cas que no tinguis el Java instal·lat, ho pots fer fàcilment amb la
següent comanda que necessitarà privilegis d'administració. Si no els
tens, demana ajuda doncs hi ha alternatives.

.. code-block:: console

    $ sudo apt install openjdk-17-jdk

En el moment d'escriure aquests continguts, ja hi ha disponibles versions més
actuals de Java. La 1.17 és el que s'anomena ``LTS`` (*Long Term Support*) el
que vol dir que gaudirà d'actualitzacions durant un bon nombre d'anys. És una
versió prou recent, i encara hi ha moltes empreses que estan lluny
d'arribar-hi. En el cas que decideixis instaŀlar una versió superior a la 1.17,
vigila no fer servir als exercicis res incompatible per la versió 1.17, ja que
no els podré corregir i, el que és pitjor per a tu, tampoc no podré posar-te
nota.

.. important::

    És important que tinguis clar que aquest **no** és un curs de Java. És un
    curs de programació i no de codificació! Només fem servir aquest llenguatge
    com a eina per aprendre els conceptes de programació que després faràs
    servir en ves-a-saber-tu-quin-llenguatge (potser ni existeix encara!).

    Les característiques particulars de Java fan que sigui una bona elecció per
    aquest curs, no pateixis. A més, la versió 1.17 és molt `moderna per la
    majoria de les empreses
    <https://newrelic.com/resources/report/2023-state-of-the-java-ecosystem>`_.
