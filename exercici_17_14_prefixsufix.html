Exercici 17_14. Prefixos i sufixos

Context

  • Carpeta de lliurament: 17_14_prefixsufix/

  • Continguts relacionats: Exercicis addicionals

  • Com lliurar-lo: instruccions

  • [✓] Exercici amb autoavaluació

Enunciat

Implementarem ara una nova versió per les utilitats de String: startsWith i endsWith.

Les versions a implementar tindran dues modalitats per permetre indicar si ha de ser extricte o no en el sentit de l'exercici anterior.

Les signatures a implementar són:

// retorna cert quan text comença amb prefix, considerant si ha de ser o no extricte
public static boolean esPrefix(String text, String prefix, boolean extricte)
// equival a esPrefix(text, prefix, true)
public static boolean esPrefix(String text, String prefix)

// retorna cert quan text finalitza amb sufix, considerant si ha de ser o no extricte
public static boolean esSufix(String text, String sufix, boolean extricte)
// equival a esSufix(text, sufix, true)
public static boolean esSufix(String text, String sufix)

Pots provar el seu funcionament amb el següent codi d'exemple:

 1 /* Aquest programa comprova el funcionament de les funció UtilSTring.esPrefix() i esSufix() */
 2 public class Main {
 3     public static void main(String[] args) {
 4         System.out.println("Text principal?");
 5         String principal = Entrada.readLine();
 6         System.out.println("Subtext?");
 7         String subtext = Entrada.readLine();
 8         System.out.println("Extricte?");
 9         boolean extricte = UtilitatsConfirmacio.respostaABoolean(Entrada.readLine());
10         System.out.printf("esPrefix(\"%s\", \"%s\") -> %b%n",
11                           principal, subtext, extricte,
12                           UtilString.esPrefix(principal, subtext, extricte));
13         System.out.printf("esSufix(\"%s\", \"%s\") -> %b%n",
14                           principal, subtext, extricte,
15                           UtilString.esSufix(principal, subtext, extricte));
16     }
17 }
