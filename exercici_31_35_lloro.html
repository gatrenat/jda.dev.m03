Exercici 31_35. El lloro

Context

  • Carpeta de lliurament: 31_35_lloro/

  • Continguts relacionats: Alguns exercicis

  • Com lliurar-lo: instruccions

  • [✓] Exercici amb autoavaluació

Enunciat

Desenvolupa un programa que vagi demanant paraules i les repeteixi fins que rebi una resposta en blanc o bé buida. És a dir, la resposta sigui un return directament o bé només espais.

Un exemple d'execució seria:

$ java Lloro
El lloro espera paraula:
hola
El lloro repeteix: hola
El lloro espera paraula:
què tal?
El lloro repeteix: què tal?
El lloro espera paraula:

Adéu

Fixa't que la resposta del lloro canvia lleugerament en finalitzar.
