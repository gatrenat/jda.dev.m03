Exercici 32_44. La utilitat split

Context

  • Carpeta de lliurament: 32_44_split/

  • Continguts relacionats: Seqüències en Java

  • Com lliurar-lo: instruccions

  • [☆] Exercici amb dificultat addicional

  • [✓] Exercici amb autoavaluació

Enunciat

Implementa una versió simplificada de la utilitat String.split()

La versió que implementarem considerarà com a separador els caracters en blanc (Character.isWhitespace()) El resultat serà sempre un array de String i disposarà de dues modalitats que permetran decidir què fer amb els espais en blanc, si incloure'ls o no en el resultat.

Les signatures a implementar són:

// retorna la seqüència de subcadenes de text separades entre caràcters en
// blanc, incloent els caracters en blanc quan inclouBlancs és cert
public static String[] separa(String text, boolean inclouBlancs)
// equival a separa(text, false)
public static String[] separa(String text)

Pots provar el seu funcionament amb el següent codi d'exemple:

 1 /* Aquest programa comprova el funcionament de la funció UtilSTring.separa() */
 2 public class Main {
 3     public static void main(String[] args) {
 4         System.out.println("Text principal?");
 5         String principal = Entrada.readLine();
 6         System.out.println("Inclou espais?");
 7         boolean inclouBlancs = UtilitatsConfirmacio.respostaABoolean(Entrada.readLine());
 8         String[] paraules = UtilString.separa(principal, extricte);
 9         System.out.printf("separa(\"%s\", %b) ->%n", principal, extricte);
10         if (paraules.length > 0) {
11             for (int i=0; i < paraules.length; i++) {
12                 System.out.printf("- \"%s\"\n", paraules[i]);
13             }
14         } else {
15             System.out.println("Cap resultat");
16         }
17     }
18 }
